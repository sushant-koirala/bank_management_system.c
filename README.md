# bank_management_system.c
A Bank Management System developed in C by using conditional statement,loops,fuctions,structure and filehandling.
Completed My First Semester Mini Project: Bank Management System in C

I developed a Bank Management System using the C programming language as part of my first-semester project.

1.Create new bank accounts

2. Display all account records

3. Search accounts by account number

4.Deposit money

5. Withdraw money

6. File handling for permanent data storage

7.Simple menu-driven user interface

🛠 Concepts Used:

• Structures

• Functions

• File Handling (bank.txt)

• Conditional Statements & Loops
